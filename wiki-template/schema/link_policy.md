# Link Policy

Use standard Markdown links as canonical links.

Good:

```markdown
[LLM-Wiki](/concepts/llm-wiki.md)
```

Do not use Obsidian-style wikilinks as canonical links.

Link meaningful concepts, methods, tools, decisions, references, syntheses, projects, datasets, and contradictions. Avoid linking generic nouns only to increase graph density.

When creating an important page, add at least one inbound link from `overview.md`, a parent page, or a related page.
