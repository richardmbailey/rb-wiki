# Query Policy

Answer from the wiki first, then raw sources only when needed.

1. Classify the request as lookup, comparison, synthesis, audit, maintenance, source-check, or contradiction-check.
2. Route first using `wiki/index.md`, frontmatter, graph, search, and the source registry.
3. Read only the narrowed candidate pages.
4. Open raw sources when exact wording, ambiguity, contradiction, or explicit user request requires it.
5. Cite relevant wiki pages and reference pages.
6. Surface uncertainty and contradiction notes.
7. Create or propose reusable synthesis pages when an answer is likely to recur.
